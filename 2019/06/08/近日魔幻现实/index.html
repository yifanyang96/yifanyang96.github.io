近日魔幻现实

发表于 2019-06-08 | 分类于 生活
年初开始有以下几件事物让我感觉有些不舒服：
  1. 流浪地球电影
  2. 5G用于远程手术
  3. 复旦大学歌唱祖国宣传片
  4. 高考全国卷1作文题
  5. 高考迷信之旗开得胜、一举高中、马到成功等

它们的共同特点是都让我产生了这样的疑问：为什么现在还有人信这个？当然这是基于我个人的价值观，但我相信有这样疑问的人肯定是非常多的。问题就是认同这些的人究竟有多少？我觉得认同感可能是从1-5逐渐递减，可是它们究竟有什么区别呢？我给你看了牺牲的崇高、复旦师生的贡献、劳动的光荣，你为什么要相信我呢，既然你觉得穿旗袍意味着旗开得胜这个命题是荒谬的？如果我的这些命题跟5G可以实现远程手术一样是一个幌子呢（推广5G仍然是好的）？你当然可以经过你的思考或者体悟去相信他们，但这种通过人们易于接受的形式传播、经过艺术加工、目的是“感动”受众的作品有能力限制你的思考，第一印象以及感性认识对人的影响是巨大的。危险的糖衣炮弹。

结论：我自己感觉最近上文定义的这种作品变多了，很多也是夹带政治私货的，我觉得应该自己保持警惕也要帮助周围人保持警惕，好的办法是讨论/辩论，例如对于流浪地球可以问自己几个问题（无论有没有答案）：牺牲自我为什么崇高？这里的牺牲自我和德国、日本的法西斯主义有什么不同？和大跃进又有什么不同？
